# DSE 中文命題寫作評分準則（10 分制操作化版本）

> 用途：供教學、校內評改及 AI 評核參照。本版本保留原有比重，但把入分條件寫得更清楚，重點是提升評分一致性，而不是取代官方評卷文件。

## 一、評分結構

各評分項目先以 `0–10` 分評定，再按比重換算為總分。

| 評分項目 | 原始分 | 換算方式 | 主要考核重點 |
| --- | --- | --- | --- |
| 內容 | 0–10 | `N x 4` | 扣題程度、立意深度、選材是否恰當、內容是否充實 |
| 表達 | 0–10 | `N x 3` | 用詞、句式、文氣、修辭、表達是否準確流暢 |
| 結構 | 0–10 | `N x 2` | 佈局、段落、層次、詳略、過渡、首尾呼應 |
| 標點 | 0–10 | `N x 1` | 標點符號是否準確並有助文意表達 |

## 二、核心評分原則

1. 先評整體，再定分數：先判斷文章大致屬於上、中、下哪一品，再在該品內選取最合適的具體分數。
2. 內容優先：是否切合題旨，是整體入分的首要依據。即使文筆優美，若明顯偏題，內容分也不宜偏高。
3. 結構不得高於內容：文章若主旨薄弱、內容失衡，即使段落形式完整，結構分亦不應高於內容分。
4. 標點獨立評分：此項只評標點運用，不評字體工整與否。
5. 以整體表現定分：不宜因一兩句佳句或個別錯失，過度拉高或拉低整體評級。
6. 邊界從嚴看主項：若作品同時呈現兩級特徵，應看主要表現及反覆出現的問題落在哪一級。
7. `0` 分定義：白卷，或作答文字大部分無法辨識，且不足以構成可評文章。

## 三、建議評改步驟

1. 通讀全文，先判斷是否扣題、是否成篇。
2. 先給「內容」分數，確定文章的大致品第。
3. 再評「表達」：看語句是否通順、準確、有變化。
4. 再評「結構」：看文章是否有清楚的開展與層次，並確保其分數不高於內容。
5. 最後評「標點」：只按標點運用準確度給分。
6. 完成四項原始分後，再按比重折算，最後才處理錯別字加分。

## 五、各項快速入分提示

### 1. 內容

- `8–10`：明確扣題，材料充分，能展現深度、見解或真切感悟。
- `5–7`：基本扣題，但內容深度、選材代表性或展開力度不足。
- `3–4`：偏題、空泛、材料薄弱，未能有效支撐主旨。
- `0–2`：嚴重離題、內容過少，或幾乎無可評主旨。

### 2. 表達

- `8–10`：用詞準確，句式靈活，修辭自然，整體文氣成熟。
- `5–7`：基本達意，尚算通順，但較平直，文采與變化不足。
- `3–4`：語病較多，句子欠順，詞語運用明顯薄弱。
- `0–2`：大量不通順或殘缺句，影響整體理解。

### 3. 結構

- `8–10`：佈局完整，層次清楚，段落安排得宜，過渡自然。
- `5–7`：文章大致成篇，但有鬆散、失衡或個別段落安排不佳的情況。
- `3–4`：脈絡不清，段落關係弱，結構鬆散。
- `0–2`：幾乎不成篇，缺乏基本章法。

### 4. 標點

- `8–10`：標點大致準確，能配合語氣、語意與停頓。
- `5–7`：錯誤稍多，但不致嚴重妨礙理解。
- `3–4`：錯誤頻繁，已影響閱讀流暢度。
- `0–2`：大量缺漏或誤用，嚴重妨礙理解。

## 六、判分提醒

- 有文采但偏題：表達可高，內容不可高。
- 內容尚可但結構散亂：結構分應明顯低於內容分。
- 材料很多但沒有剪裁：不等於內容充實，仍應下調內容分。
- 只有開頭或結尾精彩：不足以拉高整體評級。
- 只靠大量修辭堆砌而文意空泛：表達不宜判為上品。
- 標點問題不應代替語病問題處理；語句不通屬表達，標點失當才歸入標點。

## 七、錯別字與字數要求

錯別字加分按全卷總數計算，屬總分調整項，不取代四項評分本身。

| 錯別字數量（全卷總計） | 分數調整 |
| --- | --- |
| 0–1 個 | 總分 `+3` |
| 2–4 個 | 總分 `+2` |
| 5–7 個 | 總分 `+1` |
| 8 個或以上 | 不加分 |

字數建議如下：

- 建議字數：`600` 字以上。
- 不足字數：沒有硬性扣分，但篇幅過短通常會直接限制內容的充實度與結構的完整性。
- 評改時應看「是否充分展開」而不只看字數本身；若篇幅明顯不足而論述未展開，內容及結構一般不宜判高分。
